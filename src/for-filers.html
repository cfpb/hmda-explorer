Resources for HMDA filers

The FFIEC and HUD have published the following resources for financial institutions required to file Home Mortgage Disclosure Act (HMDA) data:

Announcement

Update regarding HMDA data collected in or after 2017.

Technology Preview

The Technology Preview provides resources for financial institutions preparing their systems to file HMDA data with the CFPB.

Filing Instructions Guide

Separate Filing Instructions Guides (FIG) are now available for data collected in 2017 and 2018. Each FIG includes the previously published HMDA File Specifications.
