What is HMDA?

Watch the video or read the transcript.

Want to learn more?

Coming soon from the CFPB: easy-to-use tools that allow you to filter down millions of HMDA records to get just the data set that you need. You'll be able to preview the records before you download them, or create your own aggregations (summary tables) for additional insight.

Develop your own tools Make stuff

To store and serve all this data, we're building Qu, a public API for serving the CFPB's public data sets. Qu is open-source software: you can see our work in progress on GitHub now. We encourage you to contribute to the project, and we look forward to seeing how you'll use it to bring CFPB's data into your own applications and visualizations!
