HMDA Explorer Documentation

HMDA Explorer is a single page application that provides a landing page with visualizations and serves HMDA data from the Qu API. The code can be viewed in production at http://consumerfinance.gov/hmda.

The Explorer's main functionality is contained in a number of modular JavaScript files that use event emitters and function calls to present information using filters on the home page. These JavaScript files and their comments can be viewed in the included Docco documents. Key files include the following:

  • query.js - Determines query parameters for Qu API based on filter selections
  • forms.js - Handles dependencies and data binding for form selections
  • home.js - Handles analytics, chart, and mapping calls for the HMDA Home Page
  • dom.js - Handles click events and contains a number of helper functions
  • summary-table.js - Generates summary table queries and presentation code
  • app.js - Determines initial load, order of operations, and event emitters
